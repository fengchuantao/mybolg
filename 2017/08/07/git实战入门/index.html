    github项目开发

    github我们并不陌生，常常使用它进行项目协同或者作为自己的开源项目进行发布。github一般的使用方式大致有两种。一种是自己的团队建立一个开源的项目，另一个钟是通过fork别人的项目进行项目开发。有幸的是在日常的项目工作实践中两种我都有使用过。

    github项目参与方式-fork

    我们在项目的右上角点击fork后便会有一份克隆的数据在我们的项目中，然后我们通过在自己项目进行开发，然后pr到别人的项目上，项目人cr代码后考虑是否需要将你的代码合入根项目代码既可。这个操作方式是很方便的，有利于对代码质量的把控。

    github团队型项目开发

    我们通过创建一个团队项目，然后通过对人邀请的方式加入到项目组，然后进行开发。
